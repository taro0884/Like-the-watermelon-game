# 自作スイカゲーム

## [スイカゲームとは](https://store-jp.nintendo.com/list/software/70010000043363.html)

# 概要
動画配信サイトでスイカゲームのプレイ動画を拝見し、「自分で作成できるかも」と思って制作を始めたプロジェクトになります。

# 開発環境
Unity 2020.3.30f1
